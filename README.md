# 🧠 ZihinAğı: Secure Infrastructure Gateway

## 📌 Network Security
A centralized security layer for microservice architectures. Manages traffic integrity, authentication, and system stability.

## 🚀 Security Features
- **Dynamic Rate Limiting:** Mitigates DDoS risks and API abuse.
- **Custom Timeout Policies:** Advanced network management to prevent dangling connections.
- **Centralized Logging:** Detailed audit trails for every incoming network request.

## 📌 Project Overview
ZihinAğı is a centralized, secure API gateway engineered to manage incoming network traffic, authenticate requests, and route them to appropriate microservices. It acts as the primary defense layer and load distributor for backend architectures.

## 🚀 Key Features
- **Strict Rate Limiting:** Prevents DDoS attacks and API abuse by capping the number of requests per IP.
- **Custom Timeout Handling:** Avoids dangling connections by enforcing strict DNS and network timeout policies.
- **Centralized Authentication:** Validates payload integrity before routing to internal services.
